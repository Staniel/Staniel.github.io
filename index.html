regex to check prime number

As I am taking programming and translator course this semester, I learn a lot of fun things about regular expression. Someone posts an article describing using regular expression to check prime number. Although it’s kind of time consuming like brute force, but it seems beautiful to write it in one string pattern. The post is originally here

The magic expression is /^1?$|^(11+?)\1+$/. Very cool!
